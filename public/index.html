Welcome to Expense Tracker API

About the API

This simple REST API allows you to manage users, expenses, and income through various endpoints.

Users Endpoints

Manage users with the following endpoints:

  • GET /users - Retrieve all users
  • POST /users - Add a new user
  • PUT /users/:userId - Update an existing user by user ID
  • DELETE /users/:userId - Delete a user by user ID

Expenses Endpoints

Manage expenses with the following endpoints:

  • GET /expenses - Retrieve all expenses
  • POST /expenses - Add a new expense
  • PUT /expenses/:expenseId - Update an existing expense by expense ID
  • DELETE /expenses/:expenseId - Delete an expense by expense ID

Income Endpoints

Manage income with the following endpoints:

  • GET /income - Retrieve all income
  • POST /income - Add a new income
  • PUT /income/:incomeId - Update an existing income by income ID
  • DELETE /income/:incomeId - Delete an income by income ID
